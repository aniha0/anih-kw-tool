# ANIH 勝てるKW発掘ツール

Amazon SP広告の追加キーワードを自動で発掘する Streamlit アプリです。

## できること

検索ボリュームが大きいKWではなく、**ANIHが実際に勝てる可能性が高いKW**を抽出します。

| Tier | 条件 | 意味 |
|------|------|------|
| Tier1 | オートキャンペーンで購入あり・未登録 | 最優先で追加すべきKW |
| Tier2 | オートキャンペーンでクリック5回以上・未登録 | テスト追加を推奨 |
| Tier3 | ラッコキーワードで需要あり・未登録 | 新規開拓候補 |

## 必要なファイル（3種類）

1. **Amazon 検索用語レポート** — 広告コンソール → レポート → 検索用語
2. **Amazon キャンペーンレポート** — 広告コンソール → レポート → キャンペーン
3. **ラッコキーワード CSV**（任意）— https://related-keywords.com/

## 使い方

1. アプリを開く
2. 3つのCSVをアップロード
3. 「勝てるKW抽出」ボタンをクリック
4. 結果を確認して「追加KW.csv をダウンロード」

## ローカル起動

```bash
pip install -r requirements.txt
streamlit run app.py
```
